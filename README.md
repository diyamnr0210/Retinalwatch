#RetinalWatch
RetinalWatch is a mobile + desktop app that lets users periodically photograph their own retina at home using a smartphone + clip-on ophthalmoscope lens. A logic-based computer vision pipeline extracts key biomarkers from each photo and compares them to the user's personal baseline - flagging measurable change with full transparency.
The retina shows early signs of diabetes, hypertension, MS, and Alzheimer's years before symptoms appear.
